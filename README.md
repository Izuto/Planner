# Planner

> A simple client-side weekly planner (vanilla HTML/CSS/JS).

This project is a lightweight planner that stores all data locally in your browser using LocalStorage. It's intended for personal use: you can run it locally, create tasks and to-dos, and export events as an ICS file for calendar import.

Getting started

- Open [index.html](index.html) in your browser.
- Everything you create is stored in your browser's LocalStorage under the key `wochenplaner:v1`.
- To export tasks as calendar events, use the export button which generates an `.ics` file.

Privacy & storage

- No server, no cloud — all data remains in your browser unless you explicitly export or push files to a remote git repository.
- If you clear your browser data or LocalStorage, the app data will be lost.
