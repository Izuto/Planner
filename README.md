# Wochenplaner (lokal)

Minimaler lokaler Wochenplaner (HTML/CSS/JS).

Schnellstart:

1. Öffne die Datei `index.html` im Browser (lokal)
2. Erstelle Aufgaben, To-Dos und exportiere per ICS für Google Calendar

Hinweis: Daten werden im `localStorage` des Browsers unter `wochenplaner:v1` gespeichert.
